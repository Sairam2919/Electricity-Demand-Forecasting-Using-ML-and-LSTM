Electricity demand forecasting using Benchmark, SARIMA, SARIMAX, Random Forest, Gradient Boosting, and LSTM models on the Open Power System Data dataset.

# Electricity Demand Forecasting Using Machine Learning and Deep Learning

## Project Overview

This project compares statistical, machine learning, and deep learning approaches for forecasting electricity demand using the Open Power System Data dataset.

Models implemented include:

- Mean Forecast
- Naïve Forecast
- Seasonal Naïve
- Drift Method
- SARIMA
- SARIMAX
- Random Forest
- Gradient Boosting
- Long Short-Term Memory (LSTM)

## Dataset

- Open Power System Data
- Berlin temperature data obtained from the Open-Meteo API
- The dataset used in this project is publicly available from the Open Power System Data repository:
https://data.open-power-system-data.org/time_series/

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Statsmodels
- TensorFlow / Keras

## Evaluation Metrics

- MAE
- RMSE
- MAPE

## Results

Among all the models, the LSTM model achieved the best forecasting performance with the lowest prediction errors.
